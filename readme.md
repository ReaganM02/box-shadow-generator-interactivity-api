# Box Shadow Generator | Interactivity API
Generate beautiful box-shadow CSS instantly. Customize blur, spread, offsets, colors, and preview in real time. Free, fast, and developer-friendly box shadow generator.

**Note:** I built this app on my second day of learning the Interactivity API and WordPress Block development. Expect rapid changes and ongoing improvements.
